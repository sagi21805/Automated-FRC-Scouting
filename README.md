# Installation

## Install cmake

`
sudo apt install -y g++ cmake make git libgtk2.0-dev pkg-config 
`

## Install OpenCV
`
cd ~
`

`
git clone https://github.com/rogersce/cnpy.git
`

`
cd mkdir -p build && cd build
`

`
cmake ../cnpy
`

`
make -j4
`

`
sudo make install 
`
